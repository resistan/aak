# ABT-2.5.4-Processor: 동작기반 작동 v0.21

### 🔗 References
- KWCAG 2.2: 2.5.4 동작기반 작동
- WCAG 2.2: 2.5.4 Motion Actuation (A) (Reference)

---

이 알고리즘은 기기를 흔들거나 기울이는 동작 센서를 이용한 기능이 일반적인 인터랙션(클릭, 탭 등)으로도 수행 가능한지 진단한다.

## 1. 진단 방식
- 브라우저 환경에서 센서 의존성을 기계적으로 판별하기 어려우므로, 전면 수동 검사 항목으로 운영함.

## 2. 분석 및 분류 로직 (Analysis Pipeline)
- **Rule 2.5.4 (Manual Inspection Required):** 
  - 페이지 스캔 시 항상 단일 수동 검사 리포트 생성.
  - 결과: **[검토 필요]**.
  - 가이드: "기기를 흔들거나 기울이는 동작을 요구하는 기능이 있다면, 버튼 클릭 등 정적인 입력으로도 동일한 기능을 수행할 수 있는지 점검하세요."

## 3. 최종 상태 정의 (Final Status)
1. **검토 필요:** 전문가가 직접 동작 기반 기능의 존재 여부 및 대체 수단을 확인해야 하는 상태.
