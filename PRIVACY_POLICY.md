# 개인정보 처리방침 (Privacy Policy)

A11Y Browser Tester(이하 AAK)는 사용자의 개인정보를 소중히 여기며, 투명하고 안전한 진단 환경을 제공하기 위해 최선을 다하고 있습니다. 본 확장 프로그램의 데이터 처리 방식은 다음과 같습니다.

---

## 1. 개인정보 수집 및 전송 없음
AAK는 사용자의 개인정보를 수집, 저장 또는 외부 서버로 전송하지 않습니다. 모든 웹 접근성 진단 프로세스는 사용자의 브라우저 로컬 환경 내에서만 수행됩니다.

## 2. 데이터 저장 방식
진단 결과, 전문가 소견 및 설정값은 브라우저의 로컬 스토리지(`chrome.storage.local`)에만 저장됩니다.
- 저장된 데이터는 오직 사용자의 기기 내에서만 유지됩니다.
- 어떠한 데이터도 개발자나 제3자에게 전송되지 않습니다.
- 확장 프로그램을 삭제하면 로컬에 저장된 모든 진단 데이터도 함께 삭제됩니다.

## 3. 브라우저 권한 사용 안내
AAK는 원활한 접근성 진단 기능을 제공하기 위해 다음과 같은 권한을 사용하며, 이는 오직 명시된 목적으로만 활용됩니다.

- **`tabs`**: 현재 활성화된 탭의 URL 정보를 확인하고 진단 대상을 식별하기 위해 사용합니다.
- **`sidePanel`**: 진단 결과와 전문가 도구를 상시 노출할 수 있는 사이드 패널 인터페이스를 제공하기 위해 사용합니다.
- **`storage`**: 진단 내역과 전문가 판정 결과를 로컬에 안전하게 보관하기 위해 사용합니다.
- **`scripting`**: 웹 페이지의 DOM 구조를 분석하고 접근성 위배 요소를 탐지하는 진단 엔진을 실행하기 위해 사용합니다.
- **`activeTab`**: 사용자가 요청한 시점에 해당 페이지의 접근성 정보를 분석하기 위해 사용합니다.

## 4. 문의 사항
본 개인정보 처리방침 또는 데이터 처리 방식에 대해 궁금한 점이 있으시면 프로젝트 저장소의 Issue 기능을 통해 문의해 주시기 바랍니다.
