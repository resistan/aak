# A11Y Browser Tester 사용자 가이드

A11Y Browser Tester(이하 AAK)는 한국 웹 접근성 지침(KWCAG 2.2)을 기반으로 웹 접근성 전문가의 검수 업무를 보조하기 위해 설계된 브라우저 확장 프로그램입니다. 본 가이드는 AAK의 설치부터 정밀 진단, 전문가 도구 활용 및 리포트 생성까지의 전 과정을 안내합니다.

---

## 1. 설치 및 활성화

### 확장 프로그램 설치
1. 프로젝트를 빌드하여 `dist` 폴더를 생성합니다 (`npm run build`).
2. Chrome 브라우저에서 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭하고, 프로젝트의 `dist` 폴더를 선택합니다.

### 사이드 패널 활성화
1. 브라우저 툴바에서 확장 프로그램 아이콘(퍼즐 모양)을 클릭합니다.
2. **AAK (A11Y Assistant for KWCAG)**를 찾아 고정(Pin)하거나 클릭합니다.
3. 브라우저 우측에 사이드 패널이 열리며 대시보드가 나타납니다.

---

## 2. KWCAG 2.2 자동 진단

AAK는 KWCAG 2.2의 33개 지표를 자동으로 분석하여 전문가가 판단해야 할 핵심 요소를 선별합니다.

### 진단 실행
- 사이드 패널 상단의 **진단 시작(Run Audit)** 버튼을 클릭합니다.
- 엔진이 현재 페이지의 DOM을 분석하여 지침별 위배 사항을 추출합니다.

### 결과 해석
진단 결과는 다음과 같은 상태로 표시됩니다:
- ✅ **적절 (Pass)**: 자동 진단 기준을 통과한 항목입니다.
- ❌ **오류 (Fail)**: 지침 위배가 확실시되는 항목입니다.
- ⚠️ **검토 필요 (Manual Check)**: 알고리즘만으로는 판단이 불가능하여 전문가의 육안 확인이 필요한 항목입니다.

### 전문가 판정 (Expert-in-the-loop)
- 각 진단 항목을 클릭하면 상세 내역과 위배 요소의 마크업을 확인할 수 있습니다.
- **Spotlight** 아이콘을 클릭하면 해당 요소의 위치가 화면에 강조 표시됩니다.
- 전문가는 자동 진단 결과를 바탕으로 최종 상태(적절/오류/수정권고)를 결정하고 상세 소견을 기록할 수 있습니다.

---

## 3. 전문가 전용 도구 활용

사이드 패널의 **도구바(Tools)** 탭에서 검수 효율을 높여주는 전용 도구를 사용할 수 있습니다.

### 명도 대비 분석기 (Contrast Analyzer)
- **EyeDropper** 기능을 사용하여 화면상의 전경색과 배경색을 직접 추출합니다.
- 추출된 색상을 바탕으로 KWCAG 기준(4.5:1 또는 3:1) 충족 여부를 즉시 계산합니다.

### 포커스 트래커 (Focus Tracker)
- 키보드 Tab 키 이동에 따른 초점 이동 순서를 시각화합니다.
- 논리적 초점 이동 순서가 위배되는 지점을 직관적으로 파악할 수 있습니다.

### 대체 텍스트 오버레이 (Alt-Text Overlay)
- 페이지 내 모든 이미지 요소 위에 `alt` 속성값을 오버레이로 표시합니다.
- 이미지와 대체 텍스트의 적절성을 한눈에 비교 검수할 수 있습니다.

### CSS 선형화 (Linearization)
- 페이지의 CSS를 일시적으로 제거하여 콘텐츠의 논리적 구조와 읽기 순서를 확인합니다.

---

## 4. 진단 리포트 익스포트

검수가 완료되면 결과물을 문서화하여 공유할 수 있습니다.

### 리포트 생성
- 대시보드 하단의 **리포트 내보내기(Export)** 버튼을 클릭합니다.
- **Markdown** 포맷을 선택하여 저장합니다.

### 리포트 활용
- 생성된 Markdown 파일에는 지침별 진단 결과, 전문가 소견, 위배 요소의 마크업 정보가 체계적으로 정리되어 있습니다.
- 이 파일은 GitHub Issue, Jira 티켓 또는 최종 검수 보고서의 기초 자료로 즉시 활용 가능합니다.

---

## 5. 상태 유지 및 관리
AAK는 `chrome.storage.local`을 사용하여 검수 상태를 실시간으로 저장합니다. 브라우저를 종료하거나 페이지를 새로고침해도 진행 중인 검수 내역과 전문가 소견은 안전하게 유지됩니다.
