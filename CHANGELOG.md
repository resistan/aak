## [0.7.0] - 2026-03-25

### 오류 수정
- 1.1.1 svg 대체 텍스트 판정 오류 수정
- 1.1.1 대체 텍스트가 없는 항목 선택 시 컨텍스트 뷰가 깨지는 버그 수정
- 한글 클래스네임을 영문으로 수정
- article에서 button role 삭제
- 검사 결과 화면 상단 세션 셀렉터에서 다른 세션 선택 시 해당 사이트가 열리지 않는 버그 수정

### 추가
- 자동 검사 결과에 대해 '해당없음' 처리 방식 추가(오검출 대응)
- 해당없음 항목에 대한 점수 산정 방식 추가

### 변경
- 엔진 콘솔 로그 정리: 스캔 흐름 중 출력되던 `console.log` 주석 처리 (ABT-Core.js, Processor-131.js, ABT-Connector.js)
- ABT-Core.js: 프로세서 간 메인 스레드 양보(`yieldToMain`) 추가 — 스캔 중 UI 인터랙션 보장
- 1.4.1 색에 무관한 콘텐츠 인식 프로세서 개선:
  - 배경색 단순 탐지 제거 → 형제 색상 비교 탐지로 대체 (같은 부모 안에서 서로 다른 배경색을 가진 텍스트 없는 요소 그룹 탐지)
  - 클래스명 기반 상태 요소 탐지 추가 (status, badge, dot, indicator, tag, chip, pill, state, label)
  - SVG 차트 판별 기준 변경: 크기(50x50) → 내부 도형 요소 수 5개 이상
  - canvas, .chart, [id*="chart"] 등 클래스/태그 기반 요소는 크기 조건 없이 탐지
  - 장식 요소 제외 조건 추가: aria-hidden, 인터랙티브 요소(button, a 등) 내부 자식
- 헤더 타이틀 변경: "A11Y Assistant" → "A11Y Assistant for KWCAG", 버전 번호 표시 추가
- UI 텍스트 한국어 통일: "진단 준비 완료", "진단 시작", "정밀 진단 중", "글자색/배경색", "제목 구조"

## [0.6.9] - 2026-03-19

### 오류 수정
- 2.1.1 키보드 사용 보장: 모달 케이스(Rule 3.1)가 항상 Rule 1.1로 잘못 출력되던 문제 해결
- 2.4.2 제목 제공: `"참고자료"`가 `"적절"`로 덮어씌워지던 버그 수정
- 3.3.2 레이블 제공: 다중 ID(공백 구분) 미지원 버그 수정 — 여러 ID가 지정된 경우에도 레이블 텍스트를 올바르게 합산 검증

### 변경
- 1.4.2 자동 재생 금지: 전수 조사형으로 전환. autoplay 속성 및 iframe URL 파라미터 기반 자동 판정 추가 (오류/수정 권고/검토 필요)
- 2.1.3 조작 가능: 전수 조사형으로 전환. 인라인 요소 제외 조건 확장 (inline-block, inline-flex 등 포함)
- 3.3.3 접근 가능한 인증: 전수 조사형으로 전환. auth 신호(폼 action, 링크·버튼 href·텍스트) 기반 탐지 추가. 감지 실패 시에도 전문가 확인 요청 리포트 생성
- 1.2.1 자막 제공: 전수 조사형으로 전환. 케이스 C 이중 상태 할당 버그 수정
- 1.4.1 색에 무관한 인식: 전수 조사형으로 전환
- 2.2.2 정지 기능 제공: 전수 조사형으로 전환. 동적 요소 미감지 시 참고자료 대신 검토 필요 리포트 생성
- 3.2.1 사용자 요구에 따른 실행: 전수 조사형으로 전환. 위험 요소 미감지 시 N/A 처리 유지
- 3.3.1 오류 정정: 전수 조사형으로 전환
- 4.2.1 웹 애플리케이션 접근성 준수: 전수 조사형으로 전환. 위젯 미감지 시 N/A 대신 검토 필요 리포트 생성. 위젯 발견 시 page-level 요약 리포트 중복 생성 제거
- 2.2.1 응답 시간 조절: `meta refresh` 감지 여부와 무관하게 JS 기반 타이머 수동 검토 리포트 항상 발행 (알고리즘 명세 반영)
- 2.2.2 정지 기능 제공: `[class*="slick"]` 셀렉터 누락 추가 (알고리즘 명세 반영)
- 2.3.1 번쩍임 제한: 특정 요소 감지 여부와 무관하게 전역 수동 검토 리포트 항상 발행 (알고리즘 명세 반영)
